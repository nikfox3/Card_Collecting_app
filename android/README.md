# CardStax Android Project

This is the Android native project for CardStax, generated by Capacitor.

## Quick Start

### Open in Android Studio

```bash
# From project root
npm run android:open
```

Or manually:
1. Open Android Studio
2. File → Open
3. Select the `android` folder

### Build APK

**Debug APK (for testing):**
1. Build → Build Bundle(s) / APK(s) → Build APK(s)
2. APK location: `app/build/outputs/apk/debug/app-debug.apk`

**Release APK (for distribution):**
1. Configure signing (see ANDROID_STUDIO_SETUP.md)
2. Build → Build Bundle(s) / APK(s) → Build APK(s)
3. APK location: `app/build/outputs/apk/release/app-release.apk`

## Configuration Files

- `app/build.gradle` - App build configuration
- `build.gradle` - Project build configuration
- `key.properties` - Signing configuration (create from key.properties.example)
- `app/src/main/AndroidManifest.xml` - App manifest and permissions

## Important Notes

- **Never commit `key.properties`** - It contains sensitive signing information
- **Version numbers** are in `app/build.gradle` (`versionCode` and `versionName`)
- **App name** is in `app/src/main/res/values/strings.xml`
