## Installation in Cursor

### Step 1: Open Extensions in Cursor

1. **Open Cursor**
2. **Click Extensions** icon in left sidebar (or press `Cmd+Shift+X` on Mac, `Ctrl+Shift+X` on Windows)
3. **Search for:** `PWABuilder Studio`
4. **Click "Install"**
5. **Author:** Microsoft (PWABuilder)

**Or use Command Palette:**
1. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows)
2. Type: `Extensions: Install Extensions`
3. Search: `PWABuilder Studio`
4. Install

### Step 2: Reload Cursor

After installation, **reload Cursor** to activate the extension.

## Using PWABuilder Studio in Cursor

### Access Commands

**Method 1: Command Palette**
1. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows)
2. Type: `PWABuilder`
3. Select command:
   - `PWABuilder: Audit PWA`
   - `PWABuilder: Package PWA`
   - `PWABuilder: Generate Service Worker`
   - `PWABuilder: Generate Icons`

**Method 2: Sidebar**
1. Look for **PWABuilder** icon in left sidebar
2. Click to open PWABuilder panel

**Method 3: Right-click**
1. Right-click on `index.html` or `manifest.json`
2. Select **"PWABuilder"** from context menu

## Troubleshooting

### Extension Not Found

If you can't find PWABuilder Studio:

1. **Check you're in Extensions view** (not Settings)
2. **Search exact name:** `PWABuilder Studio`
3. **Check author:** Should be "Microsoft" or "PWABuilder"
4. **Try direct link:** https://marketplace.visualstudio.com/items?itemName=PWABuilder.pwabuilder-studio

### Extension Not Working

1. **Reload Cursor** after installation
2. **Check Extensions** - Make sure it's enabled
3. **Restart Cursor** if needed
4. **Check Cursor version** - Make sure it's up to date
